##Grunt Wiredep

>Inject Bower packages into your source code with Grunt.

>[github.com/stephenplusplus/grunt-wiredep](https://github.com/stephenplusplus/grunt-wiredep)


```bash
$ bower install angular && grunt wiredep
```
